# Design Decision Records (DDR)

Human-conception evidence. Each entry is auto-drafted from the AI session **except the Rationale**,
which must be the developer's own words, and `Conceived-by: human`, which records that origin.
Referenced from code by id (e.g. `// PROVENANCE-BEGIN: HUMAN-AUTHORED ... DDR: DDR-001`).

Entry format:
```
## DDR-<n> — <title>
Date: <ISO>   Developer: <git user.name>   Trace: <work item>
Options considered: <options that were on the table>
Chosen: <the option>
Rationale: <the developer's own words — never fabricated>
Conceived-by: human
```
